                                    Object-Oriented Programming

->Concept

OOP is based on four main principles:
Encapsulation – Wrapping data and methods into a single unit (class)
Abstraction – Hiding complex implementation and showing only essential details
Inheritance – One class can inherit properties from another class
Polymorphism – One function can behave differently in different situations

->Example

For example, consider a Car class:
The class contains properties like speed and color
It also has methods like start() and stop()
If we create multiple objects like car1, car2, they can use the same structure but have different values.
 
->Real-World Use

In Java and MERN projects, to structure large applications
In mobile apps and web development
In game development, where objects represent characters, weapons, etc.
In banking systems, where accounts, users, and transactions are modeled as objects
